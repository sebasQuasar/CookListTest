
# Cooklist

## How to install

#### Clone the project:

```bash
git clone https://github.com/sebasQuasar/CookListTest.git
```

#### Install iOS project pependencies:

```bash
cd ios
pod install
```

#### Build, start packager and run on iOS:

```bash
cd ..
yarn run ios
```

or 

Open project in XCODE and hit run.
